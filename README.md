## Installation

Clone the project:

```bash
git clone https://github.com/habibsyuhada/themovie
```
Go to the project directory and install the dependencies
```bash
cd themovie
npm install
```
Build the application
```bash
npm run build
```
## Usage
Open index.html on dist folder after build
